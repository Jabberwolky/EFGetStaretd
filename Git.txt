git-Commands:
- Der Befehl git --version in der PowerShell oder im Terminal zeigt die aktuelle Git Version an.
- Der Befehl git config --list liefert eine Liste mit allen Werten in der Config.
- Der Befehl git config --global user.name "Oliver Theisen" setzt den Namen des Benutzters fest.
- Der Befehl git config --global user.email olivertheisen@gmail.com setzt die E-Mail Adresse des Beutzters fest.
